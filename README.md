This program ask the user to enter a number, and it prints out the multiplication number 1 to 10. 
Example the user enters the number 5, the program will print the number 5 multiplication.


Enter a number: 5
5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
5 x 6 = 30
5 x 7 = 35
5 x 8 = 40
5 x 9 = 45
5 x 10 = 50
